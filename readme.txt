For run this application:
1) You need to have docker on your system
2) At this directory you need to run "docker-compose up --build"

For test this application:
1) You need to have mogodb at your local system or you can manually run docker container for test. (It uses db)
2) Just run "npm run test"

Bonus:
-This project i implement typescript for coding.
-You can use "Hubx Postman.postman_collection.json" postman collection for api after run project on docker
